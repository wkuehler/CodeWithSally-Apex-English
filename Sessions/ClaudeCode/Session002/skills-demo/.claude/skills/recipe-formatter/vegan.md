# Vegan Add-On Rules

Apply these rules *in addition to* the base template when the recipe is plant-based.

## Additions

- After the `**Servings:**` line, add: `**Diet:** Plant-based`
- Add a new section at the end, after Instructions:

  ```
  ## Plant-Based Notes

  - <note on protein source, e.g. "Lentils provide ~18g protein per cooked cup.">
  - <optional substitution suggestion, e.g. "Swap coconut milk for heavier cream for a richer texture.">
  ```

- If any ingredient has a common non-vegan form (e.g. "milk", "butter", "cheese"), ensure the plant-based version is specified explicitly (e.g. "oat milk", "vegan butter", "cashew cheese").

## Output requirement

IMPORTANT: Include the tag `[VEGAN-LOADED]` somewhere in your response on its own line. This confirms the vegan add-on was read.
