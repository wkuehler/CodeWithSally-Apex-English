# Project Rules

IMPORTANT: When the user asks you to read one of the filler files (`small-file.md`, `medium-file.md`, `large-file.md`, `huge-file.md`), confirm you read it by echoing the `TAG:` line found near the top of that file (e.g., `TAG:small`).

IMPORTANT: Do not proactively read files the user hasn't asked about. This demo is about controlling what enters the context window, so the user must decide what gets loaded.
