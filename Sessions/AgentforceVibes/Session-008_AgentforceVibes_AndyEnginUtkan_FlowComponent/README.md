
# Session 008 — Agentforce Vibes with Andy Utken (Flow Component)

This folder contains the materials and supporting files used during **Session 008** of the *Agentforce Vibes* series, recorded with **Andy Utken**.

In this session, we explored vibe-coding with Agentforce to build a **Lightning Web Component (LWC)** designed to be embedded and used inside a **Salesforce Flow**.

---

## 📌 Folder Contents

### 1. `custom-general-MetaFiles-rules.md`

This is a **rule file** used to enforce consistency across Salesforce metadata.

It ensures that the correct **API version** is applied when generating or updating meta files during development.

---

### 2. `Prompt Used`

This file contains the exact **prompt** used during the live session to guide Agentforce in building the Lightning Web Component.

The goal of the prompt was to generate an LWC component that can be:

- Placed inside a Flow Screen  
- Configured dynamically  
- Used as a reusable Flow UI component

---

## 🚀 Session Goal

The main objective of this session was to demonstrate how developers can:

- Use Agentforce Vibes for rapid prototyping  
- Improve prompts iteratively  
- Build Flow-ready LWCs efficiently
